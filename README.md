# Vitamin-Deficiency-Detector

## 📌 Overview
This project detects vitamin deficiencies (A, B1, C, D) using image processing and a Convolutional Neural Network (CNN).

Users can upload an image of skin, eyes, or nails, and the system predicts the possible vitamin deficiency.

---

## 🚀 Features
- Image-based deficiency detection
- CNN model trained on dataset
- Flask-based web interface
- Fast and simple predictions

---

## 🧠 Technologies Used
- Python
- TensorFlow / Keras
- Flask
- OpenCV
- HTML, CSS

### 3. Open in browser
http://127.0.0.1:5000/

## 🧪 Model Info
- CNN-based image classification
- Input size: 224x224
- Classes:
  - Vitamin A
  - Vitamin B1
  - Vitamin C
  - Vitamin D

---

## ⚠️ Limitations
- Limited dataset
- Accuracy depends on image quality
- Not a medical diagnosis tool
